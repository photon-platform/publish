Extensions
==========

:categories: reference

We use a curated set of Sphinx extensions to enhance our documentation.

Core Extensions
---------------

*   `sphinx.ext.autodoc <https://www.sphinx-doc.org/en/master/usage/extensions/autodoc.html>`_: Include documentation from docstrings.
*   `sphinx.ext.viewcode <https://www.sphinx-doc.org/en/master/usage/extensions/viewcode.html>`_: Add links to highlighted source code.
*   `sphinx.ext.graphviz <https://www.sphinx-doc.org/en/master/usage/extensions/graphviz.html>`_: Add Graphviz graphs.
*   `sphinx.ext.mathjax <https://www.sphinx-doc.org/en/master/usage/extensions/mathjax.html>`_: Render math via JavaScript.

Community Extensions
--------------------

*   `myst-parser <https://myst-parser.readthedocs.io/>`_: Use Markdown in Sphinx.
*   `sphinx-revealjs <https://github.com/attakei/sphinx-revealjs>`_: Create presentations with Reveal.js.
*   `sphinx-carousel <https://sphinx-carousel.readthedocs.io/>`_: Create carousels.
*   `sphinx-copybutton <https://sphinx-copybutton.readthedocs.io/>`_: Add a copy button to code blocks.
*   `sphinx-design <https://sphinx-design.readthedocs.io/>`_: Add responsive web components.

Custom Extensions
-----------------

*   **photon_platform.publish.directives.collection**: A custom directive for aggregating content.
