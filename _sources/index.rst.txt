**Publish** is a consolidated repository designed to streamline and enhance the Sphinx documentation framework.

Our mission is to create tools for gathering, processing, and publishing content, with a vision of **"Simple HTML, State of the Art CSS"**.

Key Features
------------

*   **Unified Theme**: A single, customizable theme for all PHOTON platform documentation.
*   **Custom Components**: Reusable Sphinx components, such as custom directives and roles.
*   **Centralized Configuration**: A global ``conf.py`` reducing boilerplate.
