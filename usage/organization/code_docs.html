Code Documentation¶

Publish integrates with sphinx.ext.autodoc and sphinx-autoapi to automatically generate documentation from your code.

Docstring Best Practices¶

We recommend using the Google Style docstrings for clarity and readability.

                  port (int): The port number.

              Returns:
                  bool: True if connection was successful, False otherwise.
              """
              pass

run() → List[Node]

Process the collection directive.

Instead of rendering immediately, this method: 1. Discovers all relevant files. 2. Adds them to a hidden toctree so Sphinx knows about them (fixes “not in toctree” warnings). 3. Returns a PendingCollection node to defer rendering until metadata is ready.

Returns:

A list containing the PendingCollection node and a hidden toctree node.
