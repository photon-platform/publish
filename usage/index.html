Collections

The collection directive is a powerful tool for aggregating and listing content. It allows you to create dynamic indexes of your content based on metadata.

References & Citations

The glossary directive allows you to define terms and link to them easily throughout your documentation.

Math

You can include beautiful mathematical equations using LaTeX syntax, rendered by MathJax.

Diagrams

Publish supports Graphviz for creating diagrams. You can embed the DOT code directly in your RST files.

Taxonomy & Indexes

Publish automatically generates taxonomy pages for Tags, Categories, and Types based on the metadata in your RST files.

Structure & Vision

The mission of Publish is to create tools for gathering, processing, and publishing content. We believe that content structures should be versatile, capable of being transformed into many form factors.

Code Documentation

Publish integrates with sphinx.ext.autodoc and sphinx-autoapi to automatically generate documentation from your code.
