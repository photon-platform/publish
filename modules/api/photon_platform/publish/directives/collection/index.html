The subclasses of this class might not work with docutils. This class is strongly coupled with Sphinx.

run()¶

Process the collection directive.

Instead of rendering immediately, we: 1. Discover all relevant files. 2. Add them to a hidden toctree so Sphinx knows about them (fixes “not in toctree” warnings). 3. Return a PendingCollection node to defer rendering until metadata is ready.
